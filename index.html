Web Developing

A process of where you use HTML, CSS, JavaScript and ReactJS. Body, feature of individuality, and functionality of the web application. Php, MySql, and Cloud Services, to further the dynamics in the web pages, management of data, security and backup for your data.

Frontend Developing

This part is divided into, three sections, User Interface, User Experience, and Frontend Developing. Where I learned to apply my knowledge in HTML, CSS, and JavaScript. Where there is a programming language, I’m happy, thus this website is a product of my skill in HTML, CSS, and JavaScipt.

User Interface/Experience

Essentially, user interface is the accessibility of the web you created, this is the process where you put together what your client wants and vison, and what your client’s client will like. A vision within a vision as I would call it. Accessibility for the client’s persona, vision, and trade, my job is to level the accessibility of the gap with my client and my client’s client.

Software Testing

When everything was said and done, then of course you need to test the functionality and firsthand experience your web application or website. Is it working? Is it lagging? Is it crashing? Do the pages take too long to load?
